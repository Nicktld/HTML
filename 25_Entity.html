URL is used to locate documents on WWW.

URL Schemes

Below are several famous URL schemes

Scheme Description Usage
http hyper text transport protocol Start with: "http://", unencrypted
https hyper text transport protocol security Start with: "https://", encrypted
ftp file transport protocol Upload or download files with websites
file   Files on your PC 
Scheme://host.domain:port/path/filename

Descriptions

  • scheme - Define types of Internet Service, one of which is http, which is the most common.
  • host - Define domain host. The default host of http is www
  • domain - Define the name if Internet domain, for example w3school.com.cn
  • :port - Define the port of the host. The default port of http is 80
  • path - Define the path of the service machine
  • filename - Define the names of the documents and resources
