HTML elements are defined into two parts.
1.block level element: It initializes in a new row e.g. <h1>,<p>,<ul>,<table>
2.incline element: It will not start a new row e.g. <b>,<td>,<a>,<img>

<div> is a block element which starts a new row. It often comes with CSS and acts for layouts.

<span> is an inline element. It often shows up inside block elements.
